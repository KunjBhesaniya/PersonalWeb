# PersonalWeb

A fully redesigned personal portfolio website with a modern single-page layout.

## Stack
- HTML5
- CSS3 (responsive, glassmorphism-inspired UI)
- Vanilla JavaScript

## Sections
- Hero with call-to-actions and portfolio stats
- Selected projects
- Services
- About
- Contact callout

## Run locally
```bash
python3 -m http.server 8000
```
Then open `http://localhost:8000`.
